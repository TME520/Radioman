# wardriving
Scripts and stuff used for my wardriving setup.

## Setup

1. `git clone https://github.com/TME520/wardriving.git`
2. `apt install acpi gpsd festival mpg123`

## Usage

1. `cd ./wardriving`
2. `./wardriving_status.sh`

[ Ctrl + C ] to quit.
